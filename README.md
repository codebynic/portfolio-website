## 🚀 Live Demo

👉 https://codebynic.github.io/portfolio-website/

## 🛠️ Built With

* HTML5
* Tailwind CSS
* JavaScript
* Font Awesome (CDN)
* Google Fonts (Poppins)

## 🎨 Design & Development

This portfolio also reflects my work as a **UI/UX Designer**, where I design interfaces in **Figma** and create custom visual assets using **Krita** before developing them into responsive web pages.

* Fully responsive design (mobile-first approach)

* Hero introduction section
* Projects showcase with live previews and source code
* Tech stack & design tools section
* Certifications section
* Contact section
* Mobile-friendly navigation menu
