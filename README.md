# 🌐 Personal Portfolio Website

A modern, responsive personal portfolio website built to showcase my projects, skills, certifications, and journey as a Front-End Web Developer.

---

## 🚀 Live Demo

👉 https://codebynic.github.io/portfolio-website/

---

## 🛠️ Built With

- HTML5  
- Tailwind CSS  
- JavaScript  
- Font Awesome (CDN)  
- Google Fonts (Poppins)

---

## 📌 Features

- Fully responsive design (mobile-first)
- Modern dark UI with animated elements
- Hero introduction section
- Projects showcase with live links
- Tech stack / skills section
- Certifications section
- Contact section with email button
- Mobile-friendly navigation menu

This project was built as part of my front-end development journey to improve my skills in:
